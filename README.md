# NewsApp

NewsApp is a React-based web application that displays the latest news headlines from different categories such as business, entertainment, health, science, sports, and technology.

## Features

- Latest news headlines
- Category-wise news filtering
- Responsive user interface
- Loading spinner while fetching news
- Built with reusable React components
- Fast and clean UI using Bootstrap

## Technologies Used

- React.js
- JavaScript
- HTML5
- CSS3
- Bootstrap
- News API

## Project Structure

```bash
src/

 │    ├── News.js
 │    ├── NewsItem.js
 │    ├── Spinner.js
 │
 ├── App.js
 ├── index.js
```

## Installation

1. Clone the repository

```bash
git clone https://github.com/navneet2802/newsapp.git
```

2. Navigate to project folder

```bash
cd newsapp
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm start
```
